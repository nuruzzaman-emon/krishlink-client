Overview<br/>
<br/>
KrishiLink is a full-stack MERN-like platform where users can buy and sell agricultural products.<br/>
Anyone can browse all crops, but certain actions require authentication.<br/>
<br/>
(a) The platform includes:<br/>
<br/>
(b) Crop listings<br/>
<br/>
(c) User authentication<br/>
<br/>
(d) Private routes<br/>
<br/>
(e) User dashboard (My Profile, My Crops, My Interests)<br/>
<br/>
(f) Search system<br/>
<br/>
(g) Crop details page (auth required)<br/>
<br/>
<br/>
